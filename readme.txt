== Description ==

Global Address Validation for WooCommerce

The WooCommerce Address Validator plugin uses the Byteplant Address-Validator API (https://www.address-validator.net/address-online-verification-api.html) to validate and correct shipping addresses for 240+ countries world-wide. Stop wasting time and money with undelivered packages or having to reach out to customers to get the correct addresses.

= Key Features =

* Automatically validate and correct shipping addresses
* Customers get a real-time suggestion if they enter an undeliverable address
* Easy-to-use, automatically integrates with WooCommerce checkout pages without customization

The plugin requires an API Key (free, no credit card required):
https://www.address-validator.net/address-validation-online-api-get-free-trial.html

Data Protection and Privacy Policy: see the Frequently Asked Questions section.

== Installation ==

1. Unzip and upload the plugin to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin in the Settings section of the Administrator Dashboard.
4. Register for an API key (free, no credit card required):
https://www.address-validator.net/address-validation-online-api-get-free-trial.html
5. Copy and paste your API key to the Settings section of the Administrator Dashboard.
